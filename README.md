# 🛍️ Black Friday Sales Analysis

This project focuses on performing **Exploratory Data Analysis (EDA)** on a Black Friday sales dataset to uncover meaningful insights and patterns in customer purchasing behavior.

---

## 📌 Objective

The goal of this project is to:
- Understand customer demographics
- Analyze purchasing patterns
- Identify key factors affecting purchase amount

---

## 📂 Dataset

The dataset contains information such as:
- User ID
- Gender
- Age group
- Occupation
- City category
- Product categories
- Purchase amount

---

## 🧠 What I Did

In this project, I performed:

### 🔹 Data Cleaning
- Checked for missing values
- Handled null values in product categories

### 🔹 Data Exploration
- Explored dataset structure using `.info()` and `.describe()`
- Analyzed categorical and numerical features

### 🔹 Data Visualization
Used libraries like **Matplotlib** and **Seaborn** to visualize:
- Gender vs Purchase behavior
- Age group vs Purchase amount
- Occupation vs Spending
- City category analysis
- Product category trends

### 🔹 Insights Extraction
- Identified which age group spends the most
- Compared purchase behavior between genders
- Found patterns in occupation and spending
